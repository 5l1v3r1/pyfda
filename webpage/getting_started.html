PyFDA

Python Filter Design Analysis Tool

Prerequisites

The pyFDA tool is written in pure python, you just need a running 2.7.x or 3.4.x Python installation with some of the standard scientific extensions. It has been developed and tested with the following versions but will probably run with older versions as well:

  • numpy
  • scipy
  • matplotlib
  • pyQt
  • re (regular expressions)
  • xx
  • yyy

Running pyFDA

In order to get started, you just need to download the zip-file and unzip it to a directory of your choice. Next, you have to start pyfda.py and you should be fine.

Plans for the not-so near future include:

$ python pyfda
